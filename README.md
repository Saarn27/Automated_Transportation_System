# Automated_Transportation_System
Automated transportation system created with Python

Welcome to the automated system created to ease your life and minimize your screen time :)

Below, I will explain how to modify the system so it will work on Friday or on holidays.

1. Open the main.py file using Notepad++.
   <img width="820" height="491" alt="image" src="https://github.com/user-attachments/assets/f5050979-8666-49fe-b0b3-8c601ec98c4a" />

3. Search for "is_going_on_weekend_or_holiday" variable and change it to True.
   <img width="1912" height="999" alt="image" src="https://github.com/user-attachments/assets/d8f73240-f809-4301-9362-da4f0c1d9d1c" />

5. Save

Setup guide:
1. Go to Task Scheduler and create a basic task.
2. Give it a name.  (Create a Basic Task)
3. Select Weekly.  (Trigger)
   <img width="704" height="486" alt="image" src="https://github.com/user-attachments/assets/a4bb6534-8a91-4248-93cf-859c797a1184" />

5. Select the start date, start time, and the day it will repeat itself.
   <img width="696" height="485" alt="image" src="https://github.com/user-attachments/assets/5ae40db2-23d7-4039-8c13-2391b7f254d6" />

6. Select Start a program.  (Action)
   <img width="694" height="481" alt="image" src="https://github.com/user-attachments/assets/08dfefae-7836-446f-9852-8b61a9af273e" />

7. Select the program path and write it in the program/script input.
   <img width="694" height="489" alt="image" src="https://github.com/user-attachments/assets/ae081540-6576-46f9-853d-38b132783da2" />
